== (negated: !=)

When using two equals signs for JavaScript equality testing, some funky conversions take place.

=== (negated: !==)

When using three equals signs for JavaScript equality testing, everything is as is. Nothing gets converted before being evaluated.

Object.is(val1, val2)

When using Object.is() method, comparisons use SameValue proceduce introduced in ES2015. It's similar to === comparison, but it differs by treating +0 and -0 values as not equal and NaN values as equal to each other. 


Object.is(+0, -0) // false
Object.is(NaN, NaN) // true
